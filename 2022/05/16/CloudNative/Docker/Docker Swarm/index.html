概念

swarm

集群的管理和编号。docker可以初始化一个swarm集群，其他节点可以加入。（管理者和工作者）

Node

就是docker的一个节点

Service

任务，可以在管理节点或者工作节点来运行。

Task

容器内的命令，细节任务。

逻辑

命令->管理->api->调度->工作节点（创建Task容器维护创建！）
